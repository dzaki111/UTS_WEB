# UTS_WEB


Fitur Utama Aplikasi

Katalog Buku Dinamis: Menampilkan daftar buku dalam tata letak grid yang responsif.

Pencarian Interaktif: Memfilter katalog secara real-time menggunakan input pengguna.

Keranjang Peminjaman (Cart): Memungkinkan pengguna menambahkan buku yang ingin dipinjam.

Manajemen Data Persisten: Data keranjang disimpan menggunakan Local Storage untuk memastikan item tidak hilang saat halaman di-refresh.

Validasi Form: Menyediakan feedback visual yang jelas saat terjadi kesalahan input pada proses checkout atau formulir lainnya.
